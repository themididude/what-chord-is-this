# what chord is this?


A web tool designed to identify unknown chords by selecting notes on a Guitar, a Piano, or just a note selector.

## Why build this?

During songwriting, I often build chord progressions by feel, tweaking and moving things around until something clicks. That process sometimes leads to chords I can't identify or notate correctly.

So i built this tool to solve that exact problem: given a set of notes on a guitar or piano, tell me the exact chord I'm playing.

## Features
- Guitar mode: Select frets on a 6-string guitar neck with adjustable position
- Piano mode: Simple, practical two octave keyboard
- Note mode: Select the notes from a note picker

## Tech stack

- TypeScript
- React
- Vite
